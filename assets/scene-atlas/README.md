# Generated OREBIT Scene Atlas

This directory is generated by `tools/build-scene-atlas.py` from
`assets/art/scene-textures.json`. Do not edit the PNG pages or JSON metadata by
hand.

- Pages are capped at 4096x4096 for WebGL2 portability.
- Every sprite and sprite-sheet frame keeps its original pixel dimensions.
- Each packed frame has two pixels of edge extrusion to prevent sampling bleed.
- Runtime sampling remains nearest-neighbor.
- RmlUi images and font textures are not part of the scene atlas.

The JSON contains source hashes, source and atlas rectangles, page assignments,
and decoded-pixel hashes. The generated C++ table at
`src/render/SceneAtlas.generated.h` contains the package paths and compact runtime
layout.
